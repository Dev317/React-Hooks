# About the project

This project is used to learn all the common hooks used in React.js, which include:

<li> useState
<li> useRef
<li> useReducer
<li> useMemo
<li> useLayoutEffect
<li> useImperativeHandle
<li> useEffect
<li> useContext
<li> useCallBack

In the project directory, can run:

### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

The page will reload when you make changes.\
